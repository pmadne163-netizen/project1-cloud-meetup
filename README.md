# AWS-Based Meetup Project

# Project Overview

Cloud Meetup Project is a cloud-based web application developed using PHP and deployed on an AWS EC2 instance. The application allows users to create, view, update, and delete meetup events along with RSVP management. Instead of using a traditional database, meetup information is stored as JSON objects in Amazon S3, demonstrating cloud-native object storage and secure integration using IAM Roles.

# Technologies Used
PHP

Apache Web Server

AWS EC2

Amazon S3

IAM Role

AWS SDK for PHP

Linux (Amazon Linux)

JSON

## Project Features

- Create, update, view, and delete meetup events.
- Registered attendees using the RSVP feature.
- Stored meetup information as JSON objects in Amazon S3.
- Secured access AWS resources using IAM Role authentication.
- Deployed the application on AWS EC2 with Apache Web Server.
